# koa2-weibo
Node.js-Koa2框架生态实战－从零模拟新浪微博 代码
## 使用
1. 在`src/config/db.js`中设置数据库相关的配置
2. `node ./src/db/sync.js`执行数据库同步
3. `yarn dev`或者`npm run dev`
